# Rizz Generator

A fun and simple web application that generates random "rizz" quotes—charismatic pickup lines to charm anyone! Built with pure HTML, CSS, and JavaScript.

## Features

- **Random Rizz Quotes**: Click the "Generate Rizz" button to get a random pickup line from a curated list.
- **Responsive Design**: Looks great on desktop and mobile devices.
- **Lightweight**: No dependencies, just vanilla web technologies.

## How to Use

1. Clone the repository:
   ```bash
   git clone https://github.com/BinaryVortex/Rizz-Generator.git
   ```
2. Open `index.html` in your web browser.
3. Click the "Generate Rizz" button to see a new quote!

Alternatively, view it live [here](https://binaryvortex.github.io/Rizz-Generator/) (if deployed).
